# 2-D Arrays
It is similar to 2D matrices.<br>
It has 2 parts<br>
a. Rows - Horizontal Arrays in the 2D matrix. 
b. Columns - Vertical Arrays in the 2D matrix.
### Syntax
<pre>int arr[rows][columns];</pre>
